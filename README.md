# 🤯 Дофамин-канал «Вынос мозга»

Telegram-канал шок-фактов, заточенный под **максимальный органический рост**.

Главный двигатель роста в Telegram — **пересылки (forwards)**. Поэтому каждый пост
собран так, чтобы им хотелось поделиться: короткий крючок → факт → «вау»-добивка →
прямой призыв переслать другу.

## Что внутри
| Файл | Роль |
|------|------|
| `bot.py` | движок: выбор факта, дофаминовое форматирование, постинг, антидубль |
| `facts.json` | банк «вау»-фактов (сейчас 63; легко пополнять) |
| `config.json` | имя канала, хендл, варианты CTA и призывов к пересылке |
| `history.json` | что уже постили (чтобы не повторяться) — создаётся автоматически |
| `.github/workflows/post.yml` | облачный автопостинг 4×/день (работает при выключенном Mac) |

## Дофаминовые механики (зашиты в движок)
- **Петля любопытства** — заголовок обрывает мысль, мозг требует дочитать (эффект Зейгарник).
- **Микро-награда** — факт «заходит» за 5 секунд.
- **Переменное вознаграждение** — ротация 5 шаблонов заголовка и 10 эмодзи, посты не приедаются.
- **Социальный статус** — контентом хочется поделиться, чтобы казаться интересным.
- **CTA на forward** — прямой призыв переслать = рост канала.

## Быстрый старт

### 1. Создай бота и канал
1. В Telegram напиши **@BotFather** → `/newbot` → получи `BOT_TOKEN`.
2. Создай публичный канал, придумай ему `@handle`.
3. Добавь бота **администратором** канала (право «Публиковать сообщения»).

### 2. Впиши имя канала
В `config.json` замени `channel_name` и `channel_handle` на свои.

### 3. Проверь локально
```bash
# Предпросмотр без отправки
python3 bot.py --dry-run --count 5

# Живой пост (подставь свои значения)
BOT_TOKEN=123:ABC CHANNEL_ID=@my_channel python3 bot.py
```

### 4. Автопостинг в облаке (бесплатно, 24/7)
1. Создай репозиторий на GitHub, залей туда эту папку.
2. **Settings → Secrets and variables → Actions** → добавь:
   - `BOT_TOKEN` — токен из BotFather
   - `CHANNEL_ID` — `@my_channel`
3. Готово. Канал сам постит по расписанию (09:00 / 13:00 / 18:00 / 21:00 МСК).
   Можно запустить вручную: **Actions → dopamine-channel → Run workflow**.

## Пополнение банка фактов
Добавь объект в `facts.json`:
```json
{"id": "unique_id", "cat": "Категория", "title": "Крючок-заголовок",
 "fact": "1–3 предложения сути.", "wow": "Добивка, от которой мурашки."}
```
При исчерпании банка бот начинает заново с самых «старых» фактов — канал не встаёт.

## 📈 План роста (просто рост, без монетизации пока)
1. **Контент — двигатель №1.** Держим формат «крючок + вау + перешли». Пересылки растят канал сами.
2. **Посев (первые 500–1000 подписчиков):**
   - взаимопиар с каналами схожей тематики (папки, «подборки»);
   - репост лучших фактов в свои соцсети/сторис;
   - платный посев в 2–3 близких каналах (окупается ростом ядра).
3. **Ритм.** 4 поста в день, прайм-тайм 18:00–21:00 МСК даёт больше всего пересылок.
4. **Тест заголовков.** Раз в неделю смотрим, у каких постов больше forwards → такие темы усиливаем.
5. **Когда наберём массу** — включаем монетизацию (реклама/партнёрки/трафик на твои сервисы).
