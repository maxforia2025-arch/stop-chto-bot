> Как из TikTok/Telegram увести зрителя на свою площадку (TG-бот) и в продукт.
> Слой 2 воронки (§3 [AUTO_FUNNEL_PLAN.md](../../government/AUTO_FUNNEL_PLAN.md)).
> Принцип: **каждый ролик ведёт на бесплатный лид-магнит, а не «купи сейчас».** Бесплатное
> качает подписку в свою площадку, где уже идёт автопродажа ([PRODUCT.md](PRODUCT.md)).

## 1. Лид-магнит (бесплатно, выдаёт бот мгновенно)

**«100 ШОК-ФАКТОВ, которые взорвут ленту» — PDF.**
- Что внутри: 100 лучших фактов из банка (`../facts.json` = 72 готовы + 28 добить), оформленных
  карточками в неон-стиле (генерит `../cards.py`), по категориям, с «вау-добивкой».
- Зачем даём даром: забравший = подписчик своей площадки + попал в авто-цепочку допродажи.
  PDF ещё и **репостят** («смотри какой файл нашёл») → вирусный вторичный охват.
- Формат выдачи: бот отправляет файл сразу по `/start` (документ ≤ лимита Telegram) или ссылку.
- 🔴 Президент: положить готовый PDF рядом с ботом (Гелиос подготовит вёрстку из карточек).

## 2. Захват в TikTok

| Точка | Что ставим (авто после настройки) |
|---|---|
| **Bio (шапка)** | 1 строка оффера + `link in bio`. Текст: «🧠 100 шок-фактов PDF — бесплатно 👇». Ссылка = битлинк (Beacons/Taplink) → `t.me/<bot>?start=tt` |
| **Битлинк** | Один статичный линк на все ролики (TikTok не даёт кликаб. ссылки в описании) → редирект в бота с `src=tt` |
| **Конец ролика (CTA-кадр)** | 2 сек: «Ещё 100 таких — ссылка в шапке» + стрелка вверх. Зашито в раскадровку (CONTENT_QUEUE §CTA) |
| **Подпись** | Хук + мягкий CTA: «Забери PDF в шапке 👆». Без спам-ссылок в каждом коммменте (анти-бан §6) |
| **Закреп-коммент** | Свой первый коммент: «100 фактов PDF бесплатно → ссылка в профиле» (соц-док, поднимает CTR) |

## 3. Захват в Telegram-канале

| Точка | Что ставим |
|---|---|
| **Закреплённый пост** | Оффер + inline-кнопка «🎁 Забрать 100 фактов» → `t.me/<bot>?start=tg`. Главная точка захвата |
| **Описание канала** | «Шок-факты каждый день. 🎁 100 фактов PDF бесплатно → @<bot>» |
| **Подпись под постами** | Ротация CTA из `../config.json` (подписка + «перешли другу») — уже в движке `../bot.py` |
| **Периодический пост-напоминание** | Раз в ~20 постов — мягкий пост «забери PDF в закрепе» (не чаще, чтобы не выжигать) |

**Двойной двигатель TG:** forward-механики (`share_variants` в config) растят канал органически,
а закреп-кнопка переливает выросшую аудиторию в бота. Рост и захват работают параллельно.

## 4. Логика бота (слой захвата — что автоматизируем)

```
/start src=<канал>
  → приветствие голосом канала: «Стоп… что?! Держи 100 фактов 👇»
  → отправить лид-магнит (PDF) мгновенно
  → записать подписчика + source-тег (src=tt / tg / yt…) в базу
  → предложить подписку на канал (кнопка) — замкнуть на контент
  → поставить в авто-цепочку допродажи (см. ниже)
```

**Авто-цепочка допродажи (drip, без человека):**
| Когда | Сообщение бота |
|---|---|
| Сразу | Лид-магнит PDF + «нравится? тут таких тысячи» |
| +1 день | 1 бесплатный «факт дня» + мягкий тизер PRO-пака |
| +2 дня | Оффер **O8-пак «500 фактов PRO»** (кнопка оплаты Stars) — [PRODUCT.md](PRODUCT.md) |
| После покупки | Апселл **O11-клуб** (подписка), авто-чек, FAQ-автоответы |
| Не купил +5 дней | Мягкое напоминание + соц-док («уже N человек забрали») |

UTM `src=` даёт Дедалу свести конверсию по каналам: подписка → в бота → в покупку → выручка.
